# Intelligent Research Assistant

A multi-agent application that combines PDF RAG (Retrieval Augmented Generation) with live web search to help with research tasks.

## Tech Stack
- **Framework**: LangChain, LangGraph
- **Database**: ChromaDB
- **Frontend**: Streamlit
- **Model**: OpenAI GPT-4o
- **Search**: Tavily
